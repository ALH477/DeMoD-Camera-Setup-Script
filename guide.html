DeMoD Camera Setup Guide

Developed by DeMoD LLC. This guide provides step-by-step instructions for setting up and using the DeMoD Camera Setup codebase to turn your SBC into an IP camera system.

Overview

The codebase enables RTSP streaming from USB webcams using MediaMTX and FFMPEG. It supports single and multi-camera setups, with modular utilities for configuration, security checks, and more. Licensed under GPL v3 to ensure open-source improvements.

Installation

  1. Clone or download the repository.
  2. Make scripts executable: chmod +x setup.sh start.sh.
  3. Run ./setup.sh (sudo if needed). Select OS type and architecture, follow prompts.

Usage

CLI (Single Camera)

./start.sh

Follow prompts for device selection and auth. Stream at rtsp://<ip>:8554/cam.

Web Interface

python3 config.py

Access http://localhost:8000. Use JSONC for multi-cam or form for single. Submit to generate config and start server.

TUI Security Checker

python3 security_checker.py

Guided setup with security rating. Supports JSONC for multi-cam.

Security Notes

  • Use env vars for passwords in JSONC.
  • Enable auth and firewall rules.
  • Run non-root; use 'video' group.
